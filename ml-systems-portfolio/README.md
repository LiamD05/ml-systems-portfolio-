This repository presents a collection of applied machine learning systems built with an emphasis on reproducibility, evaluation, and decision-making.

Rather than focusing on isolated models, these projects demonstrate how machine learning is used as part of end-to-end systems: from data preprocessing and feature construction, to model selection, validation, and downstream decisions or policies.

The portfolio is organized to reflect a machine learning platform / systems perspective, highlighting evaluation infrastructure, representation learning, and model-driven decisioning.
